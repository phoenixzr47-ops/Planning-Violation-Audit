# Planning-Violation-Audit
A transformation of large, messy and unclear dump files with different formats into one, clear Excel file report in order to operate easily.

At telecom, each cell must operate with specific parameters such as BCCH frequency, Bandwidth, Cell ID, Location Area Code, etc... The reference for this audit are some messy, unclear dump files from different vendors (e.g. Huawei, Ericsson, Nokia). I have used the Power Query to transform this files into clear Excel File report in order to easily indicate the cells which are not operate at the parameters which they should.
